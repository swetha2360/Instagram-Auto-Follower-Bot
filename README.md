# 🤖 Instagram Auto-Follower Bot

This Python automation script uses **Selenium WebDriver** to log in to Instagram, navigate to a target account’s followers, scroll through them, and automatically follow users.

> ⚠️ **For educational purposes only.** This project was built to learn Selenium automation and was not used in a live environment.

---

## 🚀 Features

- Logs into your Instagram account  
- Opens a similar account’s followers list  
- Scrolls through followers  
- Follows users automatically  
- Handles cookie prompts and popups

---

## 🧰 Tech Stack

- Python 3  
- Selenium WebDriver  
- ChromeDriver
